INFORMAÇÕES TÉCNICAS - METRÔ DE PARIS A*
========================================

COMPILAÇÃO PARA WINDOWS:
------------------------
O executável foi compilado usando cross-compilation do Linux para Windows com:
- Target: x86_64-pc-windows-gnu
- Rust toolchain com MinGW
- Comando: cargo build --release --target x86_64-pc-windows-gnu

DEPENDÊNCIAS:
-------------
- egui/eframe 0.31.1 (GUI)
- csv 1.3.1 (leitura de dados)
- winapi 0.3 (Windows APIs)

ARQUITETURA:
------------
- Algoritmo A* implementado do zero
- Visualização em tempo real da busca
- Interface gráfica responsiva
- Carregamento dinâmico de dados CSV

ESTRUTURA DE ARQUIVOS:
---------------------
metro_paris_astar.exe          - Executável principal (≈10MB)
data/
├── tabela1_distancias_diretas.csv   - Heurísticas (distâncias em linha reta)
├── tabela2_distancias_reais.csv     - Distâncias reais entre estações
└── tabela_linhas_conexao.csv        - Informações das linhas do metrô

COMPATIBILIDADE:
---------------
- Windows 7 SP1 ou superior
- Arquitetura x86_64 (64-bit)
- Sem dependências externas (estaticamente linkado)
- OpenGL disponível (padrão no Windows)

ANÁLISE DE DEPENDÊNCIAS DLL:
----------------------------
Todas as DLLs são padrão do Windows:
✓ kernel32.dll, user32.dll, gdi32.dll (core Windows)
✓ msvcrt.dll (C runtime)
✓ OPENGL32.dll (gráficos)
✓ Outras APIs Windows padrão

TAMANHO DO PACOTE:
------------------
- Executável: ~10MB
- Dados CSV: ~100KB
- ZIP completo: ~4MB

PERFORMANCE:
------------
- Inicialização rápida (~1-2 segundos)
- Busca A* em tempo real
- Interface fluida a 60 FPS
- Baixo uso de memória (~50MB RAM)

DISTRIBUIÇÃO:
-------------
O arquivo Metro_Paris_Astar_Windows.zip contém tudo necessário.
Basta extrair e executar Executar_Metro_Paris.bat ou metro_paris_astar.exe
